# 微博热搜创意产品分析 Skill

## 简介

这是一个专门为Claude Code创建的Skill，用于分析微博热搜榜单并生成创意小商品设计推荐。

## 功能特点

✅ **自动抓取微博热搜** - 通过API获取实时热搜数据
✅ **深度背景研究** - 使用Web Search搜索社交媒体、新闻背景和用户洞察
✅ **AI智能评分** - 100分制评分系统，从4个维度评估产品潜力
✅ **创意产品生成** - 自动生成市场赛道、产品名称、目标人群等详细信息
✅ **可视化Dashboard** - 生成交互式HTML报告，按评分分级展示

## 评分体系（总分100分）

| 维度 | 分值 | 说明 |
|------|------|------|
| 可发展度 | 40分 | 市场规模、技术可行性、趋势持久性、竞争格局 |
| 有趣度 | 20分 | 创意独特性、情感吸引力、传播潜力 |
| 生活有用度 | 20分 | 日常整合度、解决问题能力、受众规模 |
| 生产容易度 | 20分 | 制造复杂度、材料可得性、小批量成本效益 |

## 使用方法

### 1. 确保Skill已安装

将此文件夹放在以下位置之一：
- `~/.claude/skills/weibo-trends-analyzer/`
- `<项目根目录>/.claude/skills/weibo-trends-analyzer/`

### 2. 微博API已内置

**好消息**：Skill已内置天API微博热搜接口，无需额外配置！

**API地址**：https://apis.tianapi.com/weibohot/index?key=4dfdf794141101d7bb8ece0294dbbc02

**数据格式**：
```json
{
  "code": 200,
  "msg": "success",
  "result": {
    "list": [
      {
        "hotword": "热搜关键词",
        "hotwordnum": "1234567",
        "hottag": "新/热/荐"
      }
    ]
  }
}
```

### 3. 调用Skill

在Claude Code中直接说：

```
分析微博热搜
```

或者更详细：

```
分析今日微博热搜并生成产品创意报告
```

Claude会自动识别并使用这个Skill，从内置API获取实时热搜数据。

## 输出结果

Skill会生成以下文件：

1. **weibo-trends-analysis-{日期}.html** - 交互式可视化Dashboard
   - 按评分分级展示（优秀≥80分，良好60-79分）
   - 包含详细的产品信息和评分分析
   - 支持排序、筛选、搜索功能

2. **weibo-trends-data-{日期}.json** (可选) - 结构化数据，便于二次处理

## Dashboard功能

生成的HTML Dashboard包含：

- 📊 **头部统计** - 分析总数、平均分、热门类别
- 🏆 **优秀产品区** (≥80分) - 金色主题，推荐优先开发
- ⭐ **良好产品区** (60-79分) - 蓝色主题，可考虑开发
- 📋 **其他产品区** (<60分) - 灰色主题，观望或需优化
- 🔍 **交互功能** - 排序、筛选、搜索
- 📈 **评分详情** - 4个维度的可视化进度条
- 📝 **背景研究** - 社交媒体、新闻、用户洞察摘要

## 工作流程

```
用户说"分析微博热搜" → 自动调用天API → 抓取实时热搜数据
→ 每个热点进行3-5次Web搜索 → AI分析生成产品创意
→ 计算评分 → 生成HTML Dashboard
```

## 故障排除

### 问题1: Skill没有自动触发
- 确认SKILL.md文件在正确路径
- 在请求中明确提到"微博热搜"或"trending"关键词

### 问题2: API返回空数据
- 检查API地址是否正确
- 确认API是否需要认证
- 尝试在浏览器中直接访问API

### 问题3: HTML中文显示乱码
- 确保浏览器使用UTF-8编码打开文件
- 检查HTML文件的charset声明

### 问题4: Web搜索结果不相关
- Claude会自动优化搜索关键词
- 可以在请求中明确指定搜索重点
